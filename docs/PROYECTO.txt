✨ CARACTERÍSTICAS IMPLEMENTADAS
════════════════════════════════════════════════════════════════

✅ ENTRADA INTERACTIVA
   • Ingresa URLs una por una
   • Coma automática (· · ·)
   • Validación de URLs
   • Contador de repositorios

✅ DETECCIÓN AUTOMÁTICA
   • GitHub (🐙)
   • GitLab (🦊)
   • Bitbucket (🪣)
   • Repositorios genéricos

✅ AUTENTICACIÓN FLEXIBLE
   • GitHub: Token PAT + OAuth
   • GitLab: Token PAT + OAuth
   • Bitbucket: App Password
   • Credenciales guardadas (~/.hydra-clone/)
   • Verificación automática

✅ CLONACIÓN CONCURRENTE
   • Máximo 8 simultáneos (configurable)
   • Soporte 10-1000+ repositorios
   • Manejo de errores inteligente
   • Continúa si falla un repo

✅ ANIMACIONES ELEGANTES
   • Panel estilizado (Idea 1)
   • Typing effect al inicio
   • Barra de progreso con puntos (· · ·)
   • Spinner durante procesamiento
   • Animación de "carga"

✅ REPORTE COMPLETO
   • Formato Markdown
   • Resumen de resultados
   • Repositorios exitosos
   • Repositorios con error
   • Estadísticas (tiempo, tamaño, etc)
   • Timestamp automático

✅ MANEJO DE ERRORES
   • Red (no connectivity)
   • Permisos (acceso denegado)
   • Repositorio no encontrado
   • Timeout (clonación lenta)
   • Continúa con siguientes

🚀 CÓMO USAR
════════════════════════════════════════════════════════════════

1️⃣  INSTALACIÓN RÁPIDA
    cd /home/dev/leo/hydra-clone
    bash install.sh

2️⃣  EJECUTAR LA HERRAMIENTA
    bash run.sh
    
    O manualmente:
    source venv/bin/activate
    python main.py

3️⃣  USAR CLONE MASTER
    • Ver pantalla de bienvenida animada
    • Ingresar URLs de repositorios
    • Detecta plataformas automáticamente
    • Verifica/pide credenciales
    • Clona masivamente
    • Genera reporte MD
